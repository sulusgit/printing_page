

#  BLYEPRINT web page 

A sleek, single-page web interface built with pure HTML, CSS, and JavaScript. This project is hosted via **GitHub Pages** directly from the `/docs` directory.

## Live Demo

**[View the Live Site here](https://sulusgit.github.io/printing_page/)**

---

## Project Structure

The project is organized to support GitHub Pages deployment using the `docs/` folder:

* **`/docs`**: Contains the production-ready files.
* `index.html`: The main entry point.
* `style.css`: All layout and design styling.
* `pictures/ and Galleria folders`: Images, icons, and fonts.


* **`README.md`**: Project documentation (this file).

##  Built With

* **HTML5**: Semantic structure.
* **CSS3**: Modern styling and responsive design.

##  How to Use

### 1. Local Development

To run this project locally, simply clone the repository and open the `index.html` file in your browser:

```bash
git clone https://github.com/sulusgit/printing_page.git
cd printing_page/docs
open index.html

```

### 2. Deployment

This project is configured for GitHub Pages. If you make changes:

1. Update the files inside the `/docs` folder.
2. Commit and push your changes:
```bash
git add .
git commit -m "Update UI features"
git push origin main

```


3. GitHub will automatically refresh the live site.
